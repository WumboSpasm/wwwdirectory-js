ABOUT

This is a JavaScript reimplementation of the World Wide Web Directory CD-ROM by Kris Jamsa and Ken Cope, which was sold alongside the book of the same name.
Released in June 1995 and boasting over 8,000 webpages, it is one of the most expansive collections of internet material predating the Internet Archive's formation in May 1996. Unfortunately, it came with the drawback of being packed tight inside Microsoft's proprietary Multimedia Viewer Book (.mvb) file format. However, with the help of some AutoHotkey scripts and many hours of furious button mashing, the issue was alleviated. This site is the result of that effort.
For each page that makes up the collection, you will find its original HTML code (without images) and a screenshot of the page (with images) as it was in the first half of 1995. Categorization and keyword metadata is also available. Each in-page link will either take you to the Wayback Machine or its entry in this archive (if it exists). You can use the viewer's sidebar controls to navigate the collection, or queries like the one seen above.
This project is not endorsed by or affiliated with Jamsa Press or any of the CD-ROM's authors. Its purpose is to enable research and bring exposure to an underrated piece of internet history. In the case that this application is rehosted on a public web server, you may find the extracted CD-ROM files here and the web frontend here.
